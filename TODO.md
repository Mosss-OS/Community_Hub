# Design Analysis: paywithlocus.com

## Colors
- **Primary**: #4101f6 (indigo/violet)
- **Background**: #000000 (black/dark theme for main)
- **Surface**: #0A0A0A to #1A1A1A
- **Text Primary**: #FFFFFF
- **Text Secondary**: #A1A1AA (muted gray)
- **Border**: #27272A
- **Accent**: #4101f6

## Typography
- **Font Family**: Inter (Google Font)
- **Font Display**: sans-serif
- **Heading weights**: 600, 700, 800
- **Body**: 400, 500

## Spacing & Layout
- **Max width**: 1200px-1400px centered
- **Padding**: 24px (mobile), 48px+ (desktop)
- **Gap between sections**: 96px-128px
- **Border radius**: 8px-16px (subtle rounding)

## UI Elements
- **Buttons**: Rounded, primary color filled, ghost variants
- **Cards**: Subtle borders, dark backgrounds
- **Inputs**: Dark backgrounds with subtle borders
- **Gradients**: Subtle purple/indigo glows

## Design Principles
1. Dark theme predominant
2. High contrast text
3. Clean, minimal interface
4. Generous whitespace
5. Subtle animations
6. Monospace for code/technical elements
